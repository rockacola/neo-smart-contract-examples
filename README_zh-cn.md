## 概述

从仓库的 `/examples` 目录获取各种各样的不同编程语言的智能合约例子。

### 求助！! 🙏

这个项目迫切需要c# 贡献者,需要如下方面的贡献者:

* C# 开发者协助翻译python例子。
* 简体中文翻译者，翻译md和源码文档。
* 文档复核者帮助优化学习曲线，改善知识共享体验。
